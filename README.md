# DeFi Tools

Suite d'outils DeFi personnels.

- **morpho-direct** — Browser de marchés Morpho Blue : supply/withdraw direct sans vaults curateurs, notes de risque A→D par collatéral (analyses versionnées dans `morpho-direct/data/risk-analyses/`)
- **position-monitor** — Monitoring de positions DeFi (health factor, funding, dépegs) avec alertes Telegram
- **position-monitor-dashboard** — Dashboard Next.js du monitor
